FIT Route Converter (Tobias Fork)

A browser based application to convert a GPX file into a FIT file and insert Turn-by-Turn Instructions.

  • Select the GPX file to be converted
  • Check the route; click on Points to add Turn Instructions
    Shortcuts After click on a point you can use keys to quickly set the turn.
    q
    slight left
    w
    straight
    e
    slight right
    a
    left
    s
    u-turn
    d
    right
    y/z
    sharp left
    x
    generic/other
    c
    sharp right
    GPX with Turns You can also use a GPX which already includes Turn-by-Turn Instructions and edit them. The Instructions need to be in the osmand format.
    You can use e.g. bikerouter.de or Brouter-Web for this.
  • Amend the average speed to your desired value
  • Download will save the course as a FIT file
  • Open the FIT on your Phone to Import with the App of your Navigation Device

Your data never leaves your computer as all processing is performed within your browser. You do not need to register to use this application. There are no adverts, no cookies, almost no use of local storage, or service workers.
